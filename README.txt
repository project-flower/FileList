FileList

選択されたファイルから一覧を作成します。

動作環境:
  Microsoft Windows 10
  .NET Framework 4.8

使い方:
  1. FileList.exe を実行します。
  2. 一覧をクリップボードにテキストとしてコピーする場合は、[Mode]グループ ボックスから、[Clipboard]を、
     一覧をファイルに保存する場合は、[File]を選択します。
  3. 一覧を作成したいファイルをエクスプローラ等から選択(複数選択可)し、FileList のウィンドウにドラッグ＆ドロップします。
  4. [Clipboard]が選択されている場合、成功すると、"クリップボードにコピーしました。"とメッセージが表示されます。
  5. [File]が選択されている場合、"名前を付けて保存"ダイアログが表示され、保存するファイルを選択し、
     [保存 (S)]をクリックすると、ファイルに保存されます。
     ファイルは、UTF-8形式で保存されます。
